# upfinzoai
Note: This project is not yet complete. Expect ongoing changes and updates.

## Prerequisites
1. Install Git
2. Install Node.js (v16+ recommended)
3. Install npm or yarn
4. Angular CLI installed globally
   
 npm install -g @angular/cli

## Project Setup
### Clone the Repository
git clone https://github.com/Krati156/krati156.github.io.git
cd krati156.github.io

### Install Dependencies
npm install
or
yarn install

## Running Project
### Development Server
ng serve 

Navigate to http://localhost:4200. The app will automatically reload if you change any source files.

### Poduction build
ng build

The build artifacts will be stored in the dist/ directory.
